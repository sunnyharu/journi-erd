5. 집계 쿼리 패턴 예시 (Presto SQL)
— 전체탭: 기간 KPI 집계 (mart_daily_order 기준)
SELECT
    COUNT(DISTINCT user_id)                                           AS buyer_cnt,
    COUNT(DISTINCT order_id)                                           AS order_cnt,
    COUNT(DISTINCT order_line_id)                                       AS order_line_cnt,
    SUM(IF(amt_rk = 1, COALESCE(krw_amount, 0), 0))                  AS gmv,              -- 결제거래액
    SUM(IF(delivery_rk = 1, COALESCE(krw_delivery_fee, 0), 0))        AS delivery_fee,      -- 배송비
    SUM(COALESCE(real_discount_amount, 0))                             AS coupon_discount     -- 쿠폰할인
FROM journi_y222.mart_daily_order
WHERE partner_id = :partner_id
  AND completed_dt BETWEEN :start_dt AND :end_dt
  -- cancel_yn 조건 없음 → 취소 포함 총 기준 (순 기준은 명시 요청 시에만)
— 아티스트탭: 아티스트별 일별 추이 (7일)
SELECT
    completed_dt,
    artist_id,
    artist_name,
    SUM(IF(amt_rk = 1, COALESCE(krw_amount, 0), 0))  AS gmv,
    COUNT(DISTINCT order_id)                          AS order_cnt
FROM journi_y222.mart_daily_order
WHERE partner_id = :partner_id
  AND completed_dt BETWEEN DATE_FORMAT(DATE_ADD('day', -6, CURRENT_DATE), '%Y-%m-%d') AND DATE_FORMAT(CURRENT_DATE, '%Y-%m-%d')
  AND artist_id IN (:artist_ids)     -- 다중선택, 미선택 시 조건 제거
GROUP BY 1, 2, 3
ORDER BY 1 DESC, gmv DESC
— 국가탭: 국가별 거래액 (트리맵 데이터 / 가입국가 기준)
SELECT
    join_country                                             AS country_code,
    join_country_nm                                          AS country_name,
    SUM(IF(amt_rk = 1, COALESCE(krw_amount, 0), 0))      AS gmv,
    COUNT(DISTINCT user_id)                                AS buyer_cnt,
    COUNT(DISTINCT order_id)                                AS order_cnt,
    ROUND(100.0 * SUM(IF(amt_rk=1, COALESCE(krw_amount,0),0))
        / NULLIF(SUM(SUM(IF(amt_rk=1,COALESCE(krw_amount,0),0))) OVER(), 0), 1) AS gmv_share_pct   -- 구성비
FROM journi_y222.mart_daily_order
WHERE partner_id = :partner_id
  AND completed_dt BETWEEN :start_dt AND :end_dt
  AND (:artist_id IS NULL OR artist_id = :artist_id)   -- 단일선택, 미선택 시 전체
GROUP BY 1, 2
ORDER BY gmv DESC
⚠️ 개발 유의사항
1. amt_rk = 1 중복제거 필수 — mart_daily_order는 order_line 단위 행이므로 주문금액(krw_amount) 집계 시 반드시 IF(amt_rk=1,...) 적용. 미적용 시 상품수만큼 중복 합산.
2. delivery_rk = 1 중복제거 필수 — 배송비(krw_delivery_fee)도 동일. 묶음 배송 시 1건만 대표.
3. 취소(cancel_yn) 기본 포함 — 취소제외 집계는 프론트에서 명시적 "순 기준" 선택 시에만. 기본값 = 전체(총 기준).
4. 실시간 탭 — mart는 T+1 배치. 당일 실시간 조회는 order_ro 원본 직접 조회 필요.
5. 국가 컬럼 분기 — 가입국가: join_country / 배송국가: receiver_country. 프론트 라디오버튼 값으로 동적 분기.
